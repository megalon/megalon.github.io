A chaotic, interactive, AI generated factory!

NASB Moveset Editor

A graphical moveset editor for Nickelodeon All-Star Brawl.
This project allowed the community to view, edit, and export custom movesets for characters in NASB!
Built with C# as a custom Unity Editor window, it runs in the Unity Editor.

The Beat Cop

THE BEAT COP (TBC) is a free drum machine made in Pure Data Vanilla 0.48 and released under the GNU GPLv3 license.
It is inspired by Teenage Engineering Pocket Operator (PO-12).
It features 16 customizable samples, 16 patterns, 15 different effects, and much more!
The project is also fully customizable through Pure Data!
